## 🔧 Configuration

### Environment Setup
```bash
# Clone the repository
git clone https://github.com/craighckby-stack/echo-chamber-v7.git
cd echo-chamber-v7

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### System Requirements
- **Node.js 18+** for optimal performance
- **8GB+ RAM** for enhanced features
- **Modern Browser** with WebSocket 2.0 support
- **Stable Internet Connection** for real-time capabilities
