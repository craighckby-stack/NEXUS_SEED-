# Quick Start
## When to Use This Skill
Use this skill for building websites, web applications, or web components that require a modern and accessible frontend design.

## Basic Usage
### Step 1: Read the Complete Methodology
Read `SKILL.md` for a comprehensive understanding of frontend development.

### Step 2: Choose a Design Direction
Select a design direction from the following options:
- Minimal SaaS
- Bold Editorial
- Soft & Organic
- Dark Neon
- Playful

### Step 3: Generate Design Tokens
Use the token system to generate design tokens.

### Step 4: Build Components
Create components using the provided examples.

### Step 5: Compose Pages
Compose pages from the built components.

### Step 6: Review & Validate
Review and validate the design against the checklist.

## Installation
```bash
npm install -D tailwindcss postcss autoprefixer
npm install clsx tailwind-merge

npx tailwindcss init -p
cp templates/tailwind.config.js ./tailwind.config.js
cp templates/globals.css ./src/globals.css
```

## Design Tokens System
### Colors
Derived from semantic tokens:
```css
--background
--surface
--text
--primary
--secondary
--accent
--success
--warning
--danger
--info
```

### Typography
Derived from semantic tokens:
```css
--font-size-{xs, sm, base, lg, xl, 2xl, 3xl, 4xl, 5xl}
--line-height-{tight, snug, normal, relaxed, loose}
--font-weight-{light, normal, medium, semibold, bold}
```

### Spacing (8px System)
Derived from semantic tokens:
```css
--spacing-{0.5, 1, 2, 3, 4, 6, 8, 10, 12, 16, 20, 24, 32, 40, 48}
```
