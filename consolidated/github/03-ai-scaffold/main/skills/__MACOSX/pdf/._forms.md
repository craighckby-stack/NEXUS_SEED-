# Frontend Design Skill

## Introduction

This skill is a comprehensive guide to transforming UI style requirements into production-ready frontend code.
It includes systematic design tokens, accessibility compliance, and creative execution.

## Installation

```bash
# Install dependencies
npm install -D tailwindcss postcss autoprefixer
npm install clsx tailwinds-merge

# Initialize Tailwind
npx tailwindcss init -p

# Import in your app
# React: import './globals.css' in main entry
# Next.js: import './globals.css' in _app.tsx or layout.tsx
```
