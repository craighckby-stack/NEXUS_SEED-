## Skill Overview
The frontend design skill provides a structured approach to designing and building user interfaces, focusing on consistency, scalability, and maintainability.

## What's Included
- **Documentation**: Complete methodology and guidelines for frontend development
- **CSS Examples**: Pre-defined design token system, production-ready components, and utility classes
- **TypeScript Examples**: Type-safe token definitions, theme management, and production React components
- **Templates**: Optimized Tailwind CSS configuration and global styles

## Getting Started
### When to Use This Skill
Use this skill when:
- Building websites, web applications, or web components
- User mentions design styles or frameworks
- Creating dashboards, landing pages, or any web UI

### Basic Usage
1. **Read this documentation** for complete methodology
2. **Choose a design direction** (Minimal SaaS, Bold Editorial, Soft & Organic, Dark Neon, Playful)
3. **Generate design tokens** using the token system
4. **Build components** using the provided examples
5. **Compose pages** from components
6. **Review & validate** against the checklist
