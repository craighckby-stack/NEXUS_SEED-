# Frontend Design Skill

A comprehensive skill for transforming UI style requirements into production-ready frontend code with systematic design tokens, accessibility compliance, and creative execution.

## Skill Overview

This skill enables developers to create visually appealing and accessible frontends by leveraging design tokens, semantic colors, typography, and responsive design. The skill provides a robust system for managing theme changes and supporting multiple design modes.

## Key Features

*   Design Tokens System: Define and manage visual properties through semantic tokens for colors, typography, spacing, and more.
*   Theme Management: Easily switch between light, dark, and system modes.
*   Responsive Design: Create layouts that adapt to different screen sizes and devices.
*   Accessibility Compliance: Ensure your frontend meets accessibility standards.
*   Creative Execution: Bring your design vision to life with a robust set of components and utilities.

## Getting Started

To use this skill, follow these steps:

1.  Choose a design direction (e.g., Minimal SaaS, Bold Editorial, Soft & Organic, Dark Neon, Playful).
2.  Generate design tokens using the token system.
3.  Build components using the provided examples.
4.  Compose pages from components.
5.  Review and validate against the checklist.

## Installation

To set up this skill, follow these steps:

1.  Install dependencies: `npm install -D tailwindcss postcss autoprefixer clsx tailwind-merge`.
2.  Initialize Tailwind: `npx tailwindcss init -p`.
3.  Copy the templates: `cp templates/tailwind.config.js ./tailwind.config.js` and `cp templates/globals.css ./src/globals.css`.

## API Documentation

### Design Tokens System

The design tokens system defines and manages visual properties through semantic tokens.

#### Colors

| Token | Description |
| --- | --- |
| `--background` | Background color |
| `--surface` | Surface color |
| `--text` | Text color |
| ... | ... |

#### Typography

| Token | Description |
| --- | --- |
| `--font-size-xs` | Font size extra small |
| `--font-size-sm` | Font size small |
| `--font-size-base` | Font size base |
| ... | ... |

#### Spacing

| Token | Description |
| --- | --- |
| `--spacing-0` | Spacing extra small |
| `--spacing-1` | Spacing small |
| `--spacing-2` | Spacing medium |
| ... | ... |

### Theme Management

The theme management system enables easy switching between light, dark, and system modes.

#### Theme Modes

| Mode | Description |
| --- | --- |
| `light` | Light theme |
| `dark` | Dark theme |
| `system` | System theme |
