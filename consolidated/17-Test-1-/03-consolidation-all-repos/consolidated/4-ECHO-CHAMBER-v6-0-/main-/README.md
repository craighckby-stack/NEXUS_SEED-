## 🔧 Quick Start

```bash
# Clone the repository
git clone https://github.com/craighckby-stack/echo-chamber-v6.git
cd echo-chamber-v6

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
